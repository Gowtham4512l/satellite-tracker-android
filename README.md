## Satellite Tracker Android App

An Android app built during Hardware Hackathon 2.0 that fetches live satellite data from a backend server and communicates it to hardware using BLE.

## Architecture
Server → Android App → BLE → Hardware Module

## Key Features
- Live satellite data fetching
- BLE communication with hardware
- Real-time parsing & display

## Tech Stack
- Kotlin
- BLE
- REST APIs
- Android SDK
